# Task 2: Installations

# OptiCrop: Smart Agricultural Production Optimization Engine

## Objective

The purpose of this task is to install and configure all the required software and tools needed to develop and run the OptiCrop project successfully.

---

# Software Requirements

- Python 3.12 or above
- Visual Studio Code
- Git
- GitHub Account
- GitHub Copilot Extension
- Python Extension for Visual Studio Code
- SQLite (Built into Python)

---

# Hardware Requirements

- Operating System: Windows 10/11
- Processor: Intel Core i3 or above
- RAM: Minimum 4 GB
- Storage: Minimum 500 MB Free Space
- Internet Connection

---

# Installation Steps

## Step 1: Install Python

1. Download Python from the official Python website.
2. Install Python.
3. Select **Add Python to PATH** during installation.
4. Verify installation using:

```bash
python --version
```

---

## Step 2: Install Visual Studio Code

1. Download Visual Studio Code.
2. Install it.
3. Open Visual Studio Code.

---

## Step 3: Install Git

1. Download Git.
2. Install Git using the default settings.
3. Verify installation using:

```bash
git --version
```

---

## Step 4: Install GitHub Copilot

1. Open Visual Studio Code.
2. Go to Extensions.
3. Search for **GitHub Copilot**.
4. Click Install.
5. Sign in using your GitHub account.

---

## Step 5: Install Python Extension

1. Open Extensions in Visual Studio Code.
2. Search for **Python**.
3. Install the extension developed by Microsoft.

---

## Step 6: Clone the Repository

Open Terminal and execute:

```bash
git clone https://github.com/TejaswiniBapanapalli/OptiCrop-Smart-Agricultural-Production-Optimization-Engine.git
```

Open the project folder in Visual Studio Code.

---

## Step 7: Verify the Installation

Execute the following commands:

```bash
python --version
```

```bash
git --version
```

Both commands should display the installed versions.
